# if문
## 개요
if 문은 조건식이다.
## 문법
if(조건1) {
    실행1;
}
조건1이 참이면 실행1이 실행이 된다.
if(조건1) {
    실행1;
} else if(조건2) {
    실행2;
}
조건1이 참이면 실행1이 실행된다. 조건 1이 참일 경우 어떠한 경우에도 실행2가 실행되지 않는다.
조건1이 거짓이고 조건2가 참일 때 실행2가 실행된다
